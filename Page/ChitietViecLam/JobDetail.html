Chi tiết công việc
Trách nhiệm
  • Thiết kế, xây dựng và duy trì các tính năng UI/UX chất lượng cao cho nền tảng thương mại điện tử hàng đầu bằng ReactJS.
  • Phối hợp với UI/UX Designer chuyển đổi bản thiết kế Figma thành mã HTML/CSS/React chính xác, thân thiện với accessibility.
  • Tối ưu hiệu suất ứng dụng: bundle size, tốc độ render, Core Web Vitals.
  • Xây dựng thư viện component dùng chung với Tailwind CSS theo nguyên lý design system.
  • Review code, hướng dẫn junior developer và đề xuất các giải pháp frontend tốt nhất.
  • Tích hợp RESTful API và GraphQL, xử lý async state với React Query hoặc SWR.
  • Viết unit test và integration test bằng Jest và React Testing Library để đảm bảo chất lượng code.
Yêu cầu
  • Tối thiểu 3 năm kinh nghiệm chuyên sâu về frontend với ReactJS.
  • Thành thạo Tailwind CSS, SCSS hoặc CSS-in-JS.
  • Hiểu sâu về JavaScript ES6+, TypeScript là lợi thế lớn.
  • Kiến thức về Web Performance, SEO và Accessibility (WCAG 2.1).
  • Có kinh nghiệm với Redux Toolkit, Zustand hoặc Jotai.
  • Làm quen với Vite, Webpack và các công cụ build.
  • Khả năng làm việc nhóm tốt, tư duy logic, giải quyết vấn đề linh hoạt.
  • Đọc hiểu tài liệu kỹ thuật tiếng Anh.
